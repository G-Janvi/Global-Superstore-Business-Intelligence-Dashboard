This project analyzes the Global Superstore dataset using Power BI to uncover key business insights across sales, profitability, customer behavior, and shipping performance.

## Project Overview
This project presents an end-to-end business analysis of a global retail company using the Global Superstore dataset. The dashboard provides insights into sales performance, profitability, customer behavior, and shipping efficiency to support data-driven decision making.

## Tools Used
- Power BI
- DAX
- Data Modeling (Star Schema)
- Kaggle Dataset

## Key Business Insights
- Identified top-performing regions and product categories
- Analyzed loss-making sub-categories and products
- Identified high-value customers and customer segments
- Evaluated shipping performance and delivery delays
- Measured year-over-year sales growth

## Dashboard Pages

### 1. Executive Overview
- Total Sales, Profit, Orders, Profit Margin
- Monthly Sales Trend
- Sales by Region, Category, Segment

### 2. Profitability Analysis
- Profit by Sub-Category
- Loss-making products
- Sales vs Profit analysis

### 3. Customer Insights
- Top Customers
- Customer Segmentation
- Customer Ranking

### 4. Shipping & Logistics
- Average Shipping Days
- Shipping Mode Performance
- Delivery analysis by region

## Key Features
- Interactive slicers
- Drill-down capability
- Conditional formatting
- Dynamic filtering
- Advanced DAX measures
